Lo que se hace con git tambien es generar repositorios locales 
Cuando estoy trabajando en una rama y llegue al final del dia lo que hago es un commit con lo que hice en el dia y eso se guarda en el repositorio donde queda comentado todo lo qye vamos haciendo en el proyecto. Cada rama tiene su propio repositorio
No podemos trabajar en el proyecto desde pc diferentes eso se puede hacer con github

Tiene tres estados git 
ADD
Commit

La rama main se llama master en git cuando se sube a github se llama main
con git init inicializo la carpeta en git esto se hace una sola vez y ya queda no tengo que volver a hacerlo todos los dias o cada vez que abro VScode
con git add index.html agrego un archivo al repositiorio

con git add . agrego todos los archivos al repo

con git status corroboro  que todos los archivos hayan sido agregados
