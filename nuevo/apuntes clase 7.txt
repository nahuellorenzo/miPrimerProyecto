Media query
es una propiedad de css los meda querie y se escriben cpn un arroba

@media screen and(min-width: 1024px){
	.div__container{
		background-color:red;
	}
	.img__messi{
		display:block;
	}
} esto significa que cuando la pantalla sea mas grande que 768px hace lo que esta aca adentro ademas hereda las propiedades anteriores al media querie no es necesario si ya esta el flex volverlo a poner por ejemplo porque ya esta lo hereda de antes si antes lo puso solamente pongo el flex-direction:row y me cambia la orientacion si antes estaba vertical
