keyframe: es una linea de tiempo donde van passando cosas y le vamos a dar un valor 

Keyframe = Animation_1
entonces sabemos que esa linea de tiempo la vamos a llamar con ese nombre animation_1

como toda linea de tiempo tiene etapas el start es el 0% y el final es el 100% de la linea de tiempo
con el keyframe le puedo decir la animacion que quiero que tenga del 0% al 25% del 25% al 50% y asi hasta el 100%

@keyframe cajita2movimiento {
	25%{
		background-color:red;
	}
	50%{
		background-color:blue;
	}
	75%{
		width:50%
	}
	100%{
		width:100%;
		background-color:pink;
	}
}

pero tambien en vez de hacer eso se le puede indicar desde y hasta que va a ser mas facil


@keyframe cajita2movimiento {
	from{
		background-color:aqua;
	}
	to{
		background-color:pink;
		width:100%;
	}
}
es como que hace desde el 50% del keyframe que cambia a la propiedad que pusimos en el TO y obviamente empieza con las propiedades del from

.cajita2{
